기반적인 함수적 이론
아래의 문법과 연산만으로 모든 수학적, 컴퓨터적 연산을 구현할 수 있다.

문법
  1. Variable - 변수
  2. Abstraction - 추상화
  3. Abstraction - 적용
연산
  1. -conversion(-변환)
  2. -reduction(-축약)

문법

  1. : 변수, 함수에서 쓰이는 매계변수를 표현

  2. : 람다 추상화, 함수를 정의. 를 변수로 받아서 을 반환
    (: 를 포함하는 식 )

  3. : 적용, 에 을 적용 (이후 -축약으로 이용)

연산

  1. : -conversion(-변환),
    이때 , 를 -equivalent(-동치) 라고 함

  2. : -reduction(-축약), 의 를 으로 대체
